# CVE Proof of Concept (PoC)

## Directory Structure
```txt
.
├── POC.js (JavaScript PoC for CVE-2019-5782)
```

## Description
This directory contains a JavaScript proof of concept (PoC) for [CVE-2019-5782](https://nvd.nist.gov/vuln/detail/CVE-2019-5782). 

## Usage
Download a [chrome version](https://www.slimjet.com/chrome/google-chrome-old-version.php) that is vulnerable to CVE-2019-5782, then run the PoC.
